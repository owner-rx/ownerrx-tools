# Brand Studio Plugin

Customizable branded content creation system for presentations, PDFs, pitch decks, and documents.

## What It Does

This plugin creates professional branded content using your company's visual identity and a choice of three content frameworks:

- **IGNITE** — For persuasion and sales (keynotes, pitch decks, marketing)
- **EDUCATE** — For teaching and training (workshops, tutorials, onboarding)
- **REPORT** — For data and analysis (reports, dashboards, quarterly reviews)

## Getting Started

After installing the plugin, run the **plugin customizer** to set up your brand:

1. Install this plugin in Cowork
2. Ask Claude to "customize the brand-studio plugin"
3. Answer questions about your brand colors, fonts, and visual style
4. Your plugin is now personalized for your business

## Commands

| Command | Description |
|---------|-------------|
| `/keynote [topic]` | Create a branded keynote presentation (12-20 slides) |
| `/workshop [topic]` | Create a branded workshop presentation (15-30 slides) |
| `/pdf-guide [topic]` | Create a branded PDF guide or workbook |
| `/pitch-deck [topic]` | Create a branded pitch deck (10-15 slides) |
| `/one-pager [topic]` | Create a branded one-pager (single page PDF) |

## Skills

| Skill | Purpose |
|-------|---------|
| brand-identity | Your brand colors, fonts, and visual style rules |
| content-frameworks | Three content frameworks (IGNITE, EDUCATE, REPORT) |
| presentation-creator | Master orchestrator that coordinates brand + framework into deliverables |

## Customization Points

The plugin uses placeholders that get replaced with your brand values during setup:

- **Brand name** — Your company or brand name
- **Colors** — Primary/secondary backgrounds, accent colors, text colors, gradients
- **Typography** — Heading and body font families
- **Visual style** — Overall design direction (e.g., "dark modern", "clean corporate")
- **Tone** — Content voice (e.g., "professional but approachable")
- **Target audience** — Default audience for your content
